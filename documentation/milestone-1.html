PWP Milestone 1

Purpose

This Project will be a website that allows the user to input data into fields.

Stretch goal: The site will then generate documents using pre-made templates that are filled out using the data input by the user.

Audience

This website is designed for professionals in the solar industry, specifically those in charge of filling out the documents that must be prepared and sent to solar customers in order for them to be able to claim their New Mexico solar tax credit. Target audience: solar company owners, management, employees, and other industry professionals.

Goal

The primary goal of this site is to enable users to fill out key customer information in designated data fields.

Stretch goal: This data, only entered once by the user, is then input into all of the required tax documents for the New Mexico State Solar Tax Credit, saving the user the time of having to fill out each form individually.
